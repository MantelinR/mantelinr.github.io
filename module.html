_create_parameter_dict :

update_depot_names :
Add 'ENTER' and 'LEAVE' suffixe to depots' names

create_vertices :
Create a dict which primary keys are opening or closing houres
and secondary keys are customers’ names.
Attributes are houre of opening or closing of the customer.

create_cutsomers :
Create a dict which primary keys are customers’ caracteristics
(demand, stop time, time window, ect.) and secondary keys are
customers’ name. Attributes are the value of each customer caracteristics.

create_depots :
Create a dict with depots' data. Primary keyrs are the caracteristics
of the depots, the secondary keys are the depots' names. Attributes
are the value of each depot's caracteristics.
